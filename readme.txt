Easily generate barcodes in your blog with a Widget.

== Description ==

The *Simple BarCode generator* use the [tec-it API](http://www.tec-it.com) to generate on-the-fly barcodes for your visitors.

To see it in action go to [Le Mas de Gaïa](https://lemasdegaia.com/category/extrait/).

Le générateur de code barre utilise [tec-it API](http://www.tec-it.com) pour généré à la demande des codes barre pour vos visiteurs. 

Pour le voir en action allez sur [Le Mas de Gaïa](https://lemasdegaia.com/category/extrait/).

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install it through the admin interface
2. Put the widget in your barside
3. Choose the title
4. Choose the label
5. DONE!

1. Téléchargez le dossier QRcode dans le répertoire /wp-content/plugins/, ou installez-le à partir de l'interface d'administration
2. Faites glisser le widget dans la barre latérale
3. Choisissez le titre
4. Choisissez le label
5. C'est fait !
